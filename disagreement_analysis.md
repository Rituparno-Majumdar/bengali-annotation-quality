# Disagreement Analysis

> This file will be completed after running `compute_agreement.py` following the blind annotation phase.

## Purpose

This document analyses every item where my annotation differed from the BLP-2023 gold label. For each disagreement, I record:
- The Bengali text
- My label and the gold label
- My reasoning at the time of annotation
- What the disagreement reveals about the annotation challenge

## Format (per item)

```
### Item [id]
**Text:** [Bengali text]
**My label:** [Positive / Negative / Neutral]
**Gold label:** [Positive / Negative / Neutral]
**My reasoning:** [Why I chose my label]
**Reflection:** [What this disagreement reveals — ambiguity, cultural context, sarcasm, etc.]
```
