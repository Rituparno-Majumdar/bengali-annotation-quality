# Bengali Annotation Quality — Calibration Against Gold Standard

[![Check agreement script](https://github.com/Rituparno-Majumdar/bengali-annotation-quality/actions/workflows/check.yml/badge.svg)](https://github.com/Rituparno-Majumdar/bengali-annotation-quality/actions/workflows/check.yml)

A professional annotation quality assessment: re-annotating a slice of the **BLP-2023 Bengali sentiment gold dataset** blind, then measuring agreement against the gold labels using Cohen's Kappa.

---

## Why This Repo Exists

Every annotation company calibrates candidates against a gold standard before assigning paid work. Typical thresholds: **85–90% accuracy** and **Kappa ≥ 0.70**.

This repo documents that calibration process openly — the dataset used, the rubric applied, the scores achieved, and an honest analysis of every disagreement.

---

## Dataset

**Source:** [BLP-2023 Task 2 — Sentiment Analysis of Bangla Social Media Posts](https://github.com/blp-workshop/blp_task2)  
**License:** [Creative Commons BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)  
**Task:** 3-class sentiment classification (Positive / Negative / Neutral) on Bengali social media text  
**Slice used:** 150 items — balanced 50/50/50 across classes  

---

## Methodology

1. Downloaded the BLP-2023 dev set (3,934 items with verified gold labels)
2. Sampled 150 items with balanced class distribution (random seed 42 for reproducibility)
3. Stripped gold labels — annotated blind using only the rubric in `rubric.md`
4. Ran `compute_agreement.py` to compute Cohen's Kappa and accuracy
5. Documented every disagreement in `disagreement_analysis.md`

---

## Files

| File | Description |
|---|---|
| `data/gold_sample.csv` | 150-item gold dataset slice (id, text, gold_label) |
| `data/annotation_sheet.csv` | Blind annotation sheet used during annotation (id, text, your_label) |
| `data/my_annotations.csv` | Completed annotations — filled in after blind annotation |
| `data/results_summary.csv` | Merged gold vs. annotation with match column (generated by script) |
| `rubric.md` | Annotation rubric: label definitions, decision rules, edge cases |
| `compute_agreement.py` | Computes Cohen's Kappa and accuracy; outputs results table |
| `results.md` | Final scores and label distribution comparison |
| `disagreement_analysis.md` | Item-by-item analysis of every disagreement |

---

## How to Run

```bash
pip install -r requirements.txt

# After completing data/my_annotations.csv:
python compute_agreement.py

# Custom paths:
python compute_agreement.py --gold data/gold_sample.csv --annotations data/my_annotations.csv
```
